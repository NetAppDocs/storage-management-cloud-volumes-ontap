=== Improved Cloud Volumes ONTAP support with Google Cloud C3 VMs
 
This Cloud Volumes ONTAP update enhances support for the latest Google Cloud C3 series VMs, enabling the following additional capabilities:
 
* Within your Cloud Volumes ONTAP deployments, you can switch VM types among different variants within the C3 family.
* The Google network service egress network tier will automatically toggle between Standard and Tier-1 depending on the instance type.
* You have the ability to add volumes to both new and existing aggregates on these VMs.
